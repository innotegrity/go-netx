## 👁️ Overview

`go-netx` provides additional functionality related to networking and HTTP transactions.

Please review the [module documentation](https://pkg.go.dev/go.innotegrity.dev/netx) for details on how to properly the functions and classes contained in this module.

## ✅ Requirements

This module is supported for Go v1.21 and later.
